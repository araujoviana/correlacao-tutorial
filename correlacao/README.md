# correlacao-passo-a-passo

Notebook simples que calcula a correlação de duas listas e mostra o passo a passo de cada operação.

```math
 r = \frac{1}{n-1}∑\bigg(\frac{x_i - \bar{x}}{s_x}\bigg)\bigg(\frac{y_i - \bar{y}}{s_y}\bigg)
```
